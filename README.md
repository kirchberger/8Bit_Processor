# 8Bit_Processor

The Idea of this project is to make an 8 bit processor with a very simplified instruction set out of just logic gates.

Current set of instructions:
| Instructions |
| :- |
| Load |
| Store |
| Add |
| Subtract |
| Compare |
| Jump |

Parts required for CPU 
| Part |
| :- |
| Arithmatic Logic Unit (ALU)|
| Control Unit |
| Registers |
| Clock |
| Bus |
| Instruction Decoder |
